### Windows

1. Run the installer.

### macOS
Download the \`.dmg\` file.
- **Apple Silicon (M1/M2/M3):** Look for the \`aarch64\` file.
- **Intel Macs:** ~~Look for the \`x86_64\` file.~~

**Note:** If you encounter a "damaged" or "cannot be opened" error, run the following command in Terminal:

```bash
xattr -cr /Applications/packvision.app
```

### Linux
Download the \`.deb\` or \`.AppImage\`.

```bash
# Example for .deb
sudo dpkg -i app_name_amd64.deb
```
